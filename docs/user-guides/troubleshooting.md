# Troubleshooting Guide

## Common Issues

### Route Calculation Fails

#### "Invalid port code" Error
**Problem**: The port code is not recognized.

**Solutions**:
1. Verify the UN/LOCODE format (5 letters)
2. Use autocomplete to select valid ports
3. Check port spelling

#### "Route calculation timeout" Error
**Problem**: Calculation took too long.

**Solutions**:
1. Reduce number of waypoints
2. Use fewer alternative routes
3. Try during off-peak hours
4. Contact support for complex routes

#### "No route found" Error
**Problem**: No valid route exists.

**Possible causes**:
- Vessel too large for ports/canals
- Geopolitical restrictions
- Invalid port combination

**Solutions**:
1. Check vessel dimensions
2. Review canal compatibility settings
3. Verify ports are operational

### Map Display Issues

#### Map Not Loading
**Solutions**:
1. Refresh the page
2. Clear browser cache
3. Disable ad blockers
4. Check internet connection

#### Route Not Displayed
**Solutions**:
1. Zoom out to see full route
2. Click "Center Route" button
3. Check layer visibility

### Login Issues

#### "Invalid credentials" Error
**Solutions**:
1. Verify email and password
2. Use "Forgot Password" if needed
3. Check Caps Lock status

#### Session Expired
**Solutions**:
1. Log in again
2. Enable "Remember Me" option
3. Check for browser cookie settings

### Performance Issues

#### Slow Page Loading
**Solutions**:
1. Clear browser cache
2. Disable browser extensions
3. Use supported browsers (Chrome, Firefox, Safari)
4. Check internet speed

#### Autocomplete Slow
**Solutions**:
1. Type at least 2 characters
2. Wait 500ms between keystrokes
3. Use UN/LOCODE for faster results
