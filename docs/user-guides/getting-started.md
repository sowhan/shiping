# Getting Started

## Welcome to Maritime Route Planning Platform

This guide will help you get started with the platform for optimal maritime route planning.

## First Steps

### 1. Create Your Account

1. Navigate to the login page
2. Click "Register" (or contact your administrator)
3. Complete the registration form
4. Verify your email address

### 2. Dashboard Overview

After logging in, you'll see the main dashboard with:

- **Route Search Panel** - Plan new routes
- **Recent Routes** - View your calculation history
- **Interactive Map** - Visualize routes globally
- **Quick Statistics** - Performance metrics

### 3. Plan Your First Route

1. **Select Origin Port**
   - Click the origin search box
   - Type port name or UN/LOCODE
   - Select from suggestions

2. **Select Destination Port**
   - Click the destination search box
   - Type port name or UN/LOCODE
   - Select from suggestions

3. **Configure Vessel Parameters**
   - Select vessel type
   - Enter vessel dimensions (length, beam, draft)
   - Set cruise speed

4. **Choose Optimization**
   - **Time** - Fastest route
   - **Cost** - Most economical
   - **Reliability** - Safest route
   - **Balanced** - Optimal trade-off

5. **Calculate Route**
   - Click "Calculate Route"
   - View results in seconds

## Understanding Route Results

### Primary Route

The recommended route shows:
- Total distance (nautical miles)
- Estimated duration
- Waypoints with ETAs
- Cost breakdown

### Alternative Routes

Compare up to 3 alternative routes:
- Different paths (e.g., via Suez vs Cape)
- Trade-offs between time and cost
- Risk comparisons

### Cost Breakdown

- **Fuel costs** - Based on vessel and distance
- **Port fees** - Entry/exit fees at waypoints
- **Canal fees** - Suez, Panama, etc.
- **Total estimated cost**

## Tips for Best Results

1. **Be Specific** - Enter accurate vessel dimensions
2. **Plan Ahead** - Set realistic departure times
3. **Compare Options** - Review alternative routes
4. **Check Weather** - Consider seasonal factors
